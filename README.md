## Prerequisites

You will need [Leiningen][1] 2.0 or above installed. To work on the ClojureScript side of things, use [Shadow-Cljs][2]

## Running

To start a web server for the application, run:

    lein run 

To start a CLI of netnavi, run:

    lein run -- -nw
